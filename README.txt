# Analyse av VL og SP

Dette er en plugin til QGIS som bygger på en analyse av lekkasjer og kloakkstopp
utført ved hjelp av FME. Resultatet av analysen blir visuelt representert i QGIS ved hjelp 
av denne pluginen. Målet med analysen er å identifisere vann rør eller kloakk rør som er 
større utsatt for lekkasjer eller kloakk stopp. Dette kan virke som et hjelpemiddel for å 
planlegge fornying av rør for å minske sannsynligheten for at lekkasjer eller kloakkstopp 
oppstår i framtiden.
